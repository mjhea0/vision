## install

1. `$ git clone git@github.com:mjhea0/node-base-advanced.git your_app`
2. `$ cd your_app`
3. `$ git clone https://github.com/visionmedia/node-jscoverage.git`
4. `$ cd node-jscoverage`
5. `$ ./configure && make && make install`
6. `$ cd ..`
7. `$ npm install`

## run

1. `$ node app`

## test

1. `$ grunt test`
2. `$ grunt coverage`
